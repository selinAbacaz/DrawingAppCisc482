# Hello !

## This is a quick tutorial on how to use my drawing app:

### There are 5 functionalities in this drawing app, 
#### 1. A handful of colors to choode from, 
#### 2. An eraser
#### 3. Brush Size changer
#### 4. reset canvas
#### 5. Save Drawing

### To begin, just pick a color, choose the size, and begin drawing with your finger ! When youre done, you can press 'save' to save it in your phone gallery or 'reset' to reset the canvas and draw again
